# `WIGOC` (a.k.a What Is Going On Chess)

A chess variant for those who don't know how to play chess.  

## Quickstart

* Learn how WIGOC is played by reading [guide.md](doc/guide.md).  
* Download WIGOC [here](itch.io/link/add-later).
